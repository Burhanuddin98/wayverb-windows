# Wayverb v1.2 — Windows Room Acoustics Simulator

> **[Go to the `wayverb-only` branch](https://github.com/Burhanuddin98/wayverb-windows/tree/wayverb-only)** for the full Wayverb v1.2 release — pre-built Windows executable, screenshots, build instructions, and documentation.

Wayverb is a hybrid waveguide + ray-tracing room acoustic simulator with GPU acceleration. It produces room impulse responses (`.wav` files) from 3D room models. Originally macOS-only, this fork runs on Windows 11 with MSYS2/MinGW-w64/NVIDIA OpenCL.

**Quick start:** Download the repo, switch to the `wayverb-only` branch, and run `wayverb/wayverb-0.0.1/wayverb-0.0.1/bin/wayverb.exe`.
